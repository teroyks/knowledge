## Start Heading on a New Page

How to make a style (e.g. _Heading 1_) automatically start a new page:

1. Move the cursor to a _Heading 1_ line.
2. Select _Format > Line & Paragraph spacing > Add page break before_.
3. Open the toolbar style selector, then go to _Heading 1 > Update ‘Heading 1’ to match_ to apply the page break automatically when you use the _Heading 1_ style.
