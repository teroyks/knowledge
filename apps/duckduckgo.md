# DuckDuckGo Tips

## Bang Shortcuts

- `!g` — search Google
- `!devto` - search http://Dev.to
- `!mdn` - search MDN docs
- `!yt` - search YouTube

## Miscellaneous

- `@` prefix: search for users or organizations
- `lorem ipsum` — generate 5 random paragraphs
