## Documenting Code

- [TSDoc](https://github.com/microsoft/tsdoc)
- [TypeDoc](https://typedoc.org) -- a documentation generator
  - parses both code and TSDoc comments
  - test -- generate docs from `src/` into `docs/`: `npx typedoc --out docs src`
