# Transposition (aka. Zip-star)

List of lists to list of zipped tuples:

```python
from pprint import pprint as pp

orig = [
    [12, 14, 15, 15, 17, 21, 22, 22, 23, 22, 20, 18],
    [13, 14, 14, 14, 16, 20, 21, 22, 22, 21, 19, 17],
    [2, 2, 3, 7, 9, 10, 11, 12, 10, 9, 8, 8]
]

pp(list(zip(*orig)))
[(12, 13, 2),
 (14, 14, 2),
 (15, 14, 3),
 (15, 14, 7),
 (17, 16, 9),
 (21, 20, 10),
 (22, 21, 11),
 (22, 22, 12),
 (23, 22, 10),
 (22, 21, 9),
 (20, 19, 8),
 (18, 17, 8)]
```
